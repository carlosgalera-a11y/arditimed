**Instrucciones de uso:**
- Añadir una fila por cada brecha detectada, incluyendo las de bajo riesgo (art. 33.1 in fine).
- Si se notificó a la AEPD, incluir el número de expediente en la última columna.
- Para cada brecha con post-mortem completo, crear `docs/post-mortems/YYYY-MM-DD-[descripcion].md` y referenciar desde esta tabla.
- Revisar este registro en cada auditoría trimestral.
